## Description
This project is a simple chat application built using Java Socket Programming and Swing GUI.

## Features
- User-friendly interface
- Send messages
- Receive messages
- Connect/Disconnect functionality
- Status updates

## Technologies Used
- Java
- Swing
- Socket Programming

## How to Run

Compile files:

javac ChatServer.java
javac ChatClient.java

Run server:

java ChatServer

Run client:

java ChatClient
